DAV Notes Generater - React Native Project (Source)
--------------------------------------------------
This is a complete React Native project skeleton with full UI screens:
- Home (choose class)
- Class screen (list chapters)
- Chapter screen (generate notes)
- Notes preview and export (TXT)
- About / Developer screen (with your photo)

Assets included:
- assets/app_icon.jpg  (your chosen app icon)
- assets/developer_photo.jpg  (your photo for About page)

How to build APK (on your computer):
1. Install Node.js, Java JDK (11 or 17), and Android Studio.
2. Install React Native CLI: npm install -g react-native-cli
3. In project folder: npm install
4. Run Metro: npm start
5. Build on Android: npx react-native run-android
6. For release APK: open android/ in Android Studio and use "Generate Signed Bundle / APK".

Note: This project is source-only and ready to be opened in Android Studio.
